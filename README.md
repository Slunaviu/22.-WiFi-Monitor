# WiFi Monitor — NW.js приложение

Красивое десктопное приложение для мониторинга WiFi сети в реальном времени.

## Что показывает

| Блок | Данные |
|------|--------|
| Сеть | SSID, BSSID, защита, шифрование, частота (2.4/5 GHz), канал, PHY-стандарт |
| Соединение | Уровень сигнала (визуальные полосы), локальный IP, шлюз, имя хоста, адаптер |
| Скорость | Скорость приёма/передачи (Mbps), задержка пинга до 1.1.1.1 |
| Устройства | Количество устройств в сети (ARP-сканирование) |

## Установка и запуск

### 1. Установить NW.js

Скачать с [nwjs.io](https://nwjs.io/) (рекомендуется SDK-версия для отладки).

### 2. Запуск

```bash
# Положите файлы в одну папку:
#   package.json
#   index.html
#   wifi.js

# Запустить через NW.js:
/path/to/nw .

# Или на Windows:
nw.exe .
```

### 3. Структура файлов

```
wifi-monitor/
├── package.json   # Конфиг NW.js (размер окна, без рамки и т.д.)
├── index.html     # UI интерфейс
└── wifi.js        # Node.js модуль сбора данных о сети
```

## Поддерживаемые ОС

| ОС | Метод сбора данных |
|----|--------------------|
| Windows | `netsh wlan`, `arp -a`, `route print` |
| macOS | `airport`, `ifconfig`, `arp -a` |
| Linux | `nmcli`, `iwconfig`, `arp -n`, `ip route` |

## Особенности

- **Автообновление** каждые 30 секунд
- **Кнопка обновить** вручную
- **Frameless окно** — перетаскивается за верхнюю панель
- Кнопки закрыть / свернуть в стиле macOS

## Примечание

Для точного подсчёта устройств в сети на Windows может потребоваться запуск от администратора.
